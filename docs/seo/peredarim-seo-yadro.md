# Семантическое ядро и SEO-проработка — peredarim.ru

> Обновлено под вводные: 10 городов РФ · только самовывоз · SEO + Директ · блога пока нет.

## 1. Анализ лендинга

**Продукт.** C2C-маркетплейс перепродажи свежих букетов. Человек, которому подарили цветы, выставляет букет за полцены; покупатель рядом забирает его в 2–3 раза дешевле магазина. Сделка через эскроу, комиссия 5% с продажи, есть приложение с push о свежих букетах поблизости.

**Аудитория — два сегмента:**
- **Покупатели:** охотники за недорогими свежими цветами (на свидание, в подарок, себе домой), экономные, экосознательные.
- **Продавцы:** те, кому подарили ненужный букет, не хотят выбрасывать.

**Гео.** 10 крупнейших городов РФ: Москва, СПб, Новосибирск, Екатеринбург, Казань, Нижний Новгород, Челябинск, Красноярск, Самара, Уфа.

**Логистика.** Только самовывоз. Доставки нет. Это смещает упор семантики на гиперлокальность («рядом», «у метро», «самовывоз»).

**УТП:** цена в 2–3 раза ниже магазина · свежесть с меткой «Сегодня» · безопасная сделка (эскроу) · букет рядом, самовывоз · бесплатная публикация, комиссия 5% только при продаже.

## 2. Стратегические выводы (читать до таблицы)

1. **Спрос на «передаренные букеты» не сформирован.** Брендовые и категорийные запросы ≈0. Стратегия строится на перехвате существующего спроса «дешёвые/свежие цветы рядом», а не на названии категории.

2. **Самовывоз-only отрезает самый денежный спрос.** «Доставка цветов недорого» — крупнейший коммерческий кластер, но без доставки ты его не обслужишь. Гнать его в Директ = слив бюджета и низкий показатель качества. «Доставку» убрал из ядра и занёс в минус-слова. Делаю ставку на «рядом/у метро/самовывоз», объём меньше, зато конверсия честная. **Запуск доставки — рычаг №1 для роста, основные деньги там.**

3. **Гео в Директе закрывается таргетингом по регионам, а не словами с городом.** Города в ключах («дешёвые цветы москва») режут охват и почти всегда избыточны при региональном таргетинге. Город в ключах нужен только для SEO-страниц (Title/H1 гео-лендингов), не для Директа.

4. **Без блога инфо-кластеры (5–6) сажать некуда.** Для C2C-маркетплейса узкое место — предложение (продавцы). Инфо-контент «куда деть подаренный букет» — самый дешёвый канал привлечения продавцов с почти нулевой конкуренцией. **Рекомендую мини-блог из 3–5 статей.** Пока блога нет — кластеры 5–6 помечены «на паузе».

**Приоритет:** Кластер 1 и 2 (деньги, самовывоз) → Кластер 3 (повод) → Бренд (вдолгую) → Кластеры 5–6 только после запуска блога.

## 5. Минус-слова

**Самовывоз-only — вычитать всё про доставку (для Директа обязательно):**
доставка, доставкой, доставку, доставка цветов, курьер, привезти, привезут, на дом, с доставкой

**Не тот товар/услуга:**
искусственные, мыло, мыльные, из конфет, из шаров, фоамиран, вязаные, бумажные, своими руками, мастер класс, семена, рассада, саженцы, комнатные, горшечные, букет из фруктов, из носков, из денег

**Не та интенция (картинки/инфо):**
картинки, фото, обои, нарисовать, рисунок, раскраска, значение, к чему, приметы, гадание, стихи, открытка

**Не тот бизнес-сегмент:**
оптом, франшиза, бизнес план, открыть цветочный, поставщики, оптовая база, для бизнеса, оформление зала, свадебное оформление, курсы флористики, обучение

**Халявщики (вычитать из покупательских групп):**
бесплатно, даром, бесплатные

> «на дом» не убьёт «себе домой» (разные словоформы), безопасно. Когда запустите доставку — уберите первый блок минус-слов и верните кластер доставки.

## 6. Рекомендации по размещению

### Мета-теги главной (заменить)

Сейчас Title = «Передарим — вторая жизнь подаренных букетов» — бренд-first, теряет коммерческий спрос.

- **Title:** `Свежие букеты в 2–3 раза дешевле магазина | Передарим`
- **Description:** `Свежие букеты рядом с вами в 2–3 раза дешевле цветочного. Самовывоз у дома, безопасная сделка с защитой денег. Заберите букет за полцены или передайте свой.`
- **H1:** оставить текущий — `Свежие букеты в 2–3 раза дешевле цветочного магазина`.

### Распределение по зонам

| Зона | Что размещать |
|---|---|
| Title / H1 главной | Кластер 1 + 2: «свежие букеты», «дешевле», «рядом» |
| H2 / подзаголовки | «Свежие букеты рядом, прямо сейчас», «букет за полцены, самовывоз», «безопасная сделка» |
| Текст блоков | органично вплетать Кластер 1, 2, 3; без переспама |
| Гео-страницы (×10) | `/moskva`, `/spb`, `/novosibirsk` … Title `Дешёвые свежие букеты в {городе} — самовывоз рядом` |
| Карточка букета | Title `{Цветы} за {цена} ₽ рядом, {район} — самовывоз | Передарим`; alt фото `свежий букет пионов недорого, {район}` |

### Что создать под SEO
1. **10 гео-страниц по городам запуска** — главный рычаг органики, шаблон с автоподстановкой города и района.
2. **Страница «Безопасная сделка»** (в футере ссылка есть) — раскрыть под кластер доверия, снимает возражение «обман».
3. **Шаблон alt** для каталога заложить сразу, пока каталог пуст.
4. **Мини-блог (рекомендую)** — без него кластеры 6–7 не реализовать. 3–5 статей под продавцов: «Куда деть подаренный букет вместо мусорки», «Что делать с цветами после праздника», «Как продлить жизнь срезанному букету». Прогрев supply-стороны, дешёвый трафик.
